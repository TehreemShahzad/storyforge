# 📚 StoryForge

StoryForge is an AI-powered creative writing platform that enables users to write, improve, publish, and explore stories using Artificial Intelligence.

---

## ✨ Features

- 🔐 User Authentication (Firebase)
- ✍️ AI Story Generation
- 📝 AI Rewrite & Improve Stories
- 📖 Read Published Stories
- ❤️ Like & Bookmark Stories
- 📂 Story Categories
- 👤 User Profiles
- ☁️ Cloud Firestore Database
- 📸 Firebase Storage
- 🔔 Push Notifications
- 🌙 Modern Flutter UI

---

## 🛠 Technology Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Firebase Cloud Messaging
- Provider (State Management)
- OpenAI API

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Firebase Project
- Android Studio or VS Code
- Git

### Installation

1. Clone the repository:

```bash
git clone https://github.com/TehreemShahzad/storyforge.git
```

2. Open the project:

```bash
cd storyforge
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```
